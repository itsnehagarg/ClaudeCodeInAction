# ClaudeCodeInAction
Claude Code In Action by developing production ready applications

1. Install Claude code in local system

https://code.claude.com/docs/en/quickstart

Windows CMD:

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
